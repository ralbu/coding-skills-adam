
# Exercise 5
Enter an integer number `N`
Find the largest number which can divide `N` by `3`.\
When N is less than 3 then the result is 0.\
When N is 3 then the result is 3.

**Example**

```
Input: 6
Output: 6

Input: 7
Output: 6

Input: 55
Output: 54
```
